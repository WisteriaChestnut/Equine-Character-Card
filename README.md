# Equine Character Card

一个在浏览器中运行的马匹角色卡生成器，用于整理现实赛马、拟人角色或架空马匹的基本资料、外貌设定、人际关系、比赛履历与五代血统，并导出为可独立打开和分享的 HTML 页面。

> 当前版本：Preview 0.2  
> 在线体验：[马匹角色卡片生成器](https://wisteriachestnut.github.io/Equine-Character-Card/%E8%A7%92%E8%89%B2%E5%8D%A1%E7%94%9F%E6%88%90%E5%99%A8-Preview0.2)

## 功能

- **基础资料**：英文名、中文名、性别、毛色、出生日期与简介
- **外貌设定**：身高、发色、眸色及可选身体数据
- **单位换算**：英制与公制身高、尺寸数据双向换算
- **人际关系**：自由添加关系类型、姓名和昵称
- **比赛履历**：按赛季记录赛事、级别、马场、途程、名次、奖金及备注
- **自动统计**：汇总各赛季的出赛数、冠亚季次数与奖金
- **五代血统**：编辑完整五代血统表，区分父系与母系
- **近交提示**：标记父母两系中的重复祖先，并估算近交系数
- **实时预览**：录入内容时同步生成角色卡效果
- **独立导出**：预览、下载或复制完整 HTML，无需额外资源即可打开
- **HTML 导入**：读取此前导出的角色卡，继续编辑可识别的数据

## 快速开始

本项目没有构建步骤或后端依赖。

1. 打开在线版本，或下载仓库中的 `角色卡生成器-Preview0.2.html`。
2. 使用现代浏览器直接打开该文件。
3. 在左侧录入角色资料，在右侧查看实时预览。
4. 完成后选择以下任一方式保存：
   - **预览**：在新标签页打开生成结果；
   - **下载**：保存为独立的角色卡 HTML；
   - **复制**：将完整 HTML 源码复制到剪贴板。

需要继续编辑时，点击 **导入 HTML** 并选择此前生成的文件。

## 使用说明

### 比赛记录

比赛履历按赛季组织。每个赛季可以填写年度叙述、荣誉，并添加任意场次的比赛记录。生成器会依据 `Win`、`2nd`、`3rd` 等名次自动统计胜、亚、季军次数。

### 五代血统与近交提示

血统编辑器包含本马及五代以内祖先。相同名称同时出现在父系和母系时，页面会高亮对应单元格，并按照其在血统表中的代数估算近交系数。

该数值适合角色设定和快速参考，不应替代专业血统数据库或完整的遗传学计算。

### 导出的角色卡

下载所得文件已经内嵌样式与内容，可以离线打开、单独托管或直接分享，不依赖本仓库中的其他文件。

## 数据与隐私

生成器为纯前端页面，当前版本的资料录入、预览、导入与导出均在本地浏览器中完成，不需要注册账号，仓库代码中亦未配置数据上传服务。

导出的 HTML 会直接包含你填写的全部内容。公开分享前，请自行检查其中是否含有不希望披露的信息。

## 当前版本说明

Preview 0.2 仍属于预览版本。HTML 导入通过解析已生成页面来恢复内容；若手动修改导出文件的结构、类名或文本格式，部分字段可能无法正确读回。重要设定建议另外保留备份。

## 技术实现
